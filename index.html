Deplink Manual

DepLink is an interactive tool that enables users to match gene knockouts and drug treatments that induce similar changes in cell viability and gene expression profiles.
  • Deplink aims to create an interactive and comprehensive link between genes, drugs, and their perturbations on cancer cell lines
  • Users can start their search from a gene of interest, a set of genes, a drug of interest, or SMILES code of a new drug.
  • Users can rank, interpret, and visualize effects of gene/drug perturbations using our interactive figures and annotated data tables.

Data Source and Preprocessing

This section describes in brief the data sources and pre-processing methods used to organize the data.

Cell Line Annotations


The access cell line information sample_info.csv file from DepMap Public 21Q3 was downloaded. The 21Q3 version of this file was used as it encompassed information of 2944 cell line IDs. This file was used to identify cell line IDs based on tumor type and 29 unique primary tumor type were identified for DepLink.

Broad Data


CRISPR_gene_effect.csv was downloaded and preprocessed to generate R data frames with rows corresponding to gene symbols and columns corresponding to unique cell line IDs. We then excluded NA or failed cell lines and edited the rows to only include gene symbols.








Sanger Data


gene_effect.csv was downloaded and preprocessed similarly to the Broad data (removing invalid column and changing the rows to only include gene symbols) In addition, the data was intersected to only include common gene ids among both data files.







GDSC Data


From GDSC, dose response expressing cell viabilities from drug compounds in both GDSC1 and GDSC2 were downloaded. This file was preprocessed to generate R data frames for log10-transformed half maximal inhibitory (IC50) values, where rows correspond to unique cell lines IDs and columns corresponding to 13-digit drug BRD IDs. If GDSC1 and GDSC2 reported findings on the same drug, IC50 values from one was selected for representational purposes.



PRISM Data


From PRISM dataset only chemically perturbed cell viability screens expressed via logfold change values were downloaded. This file was preprocessed to generate R data frame with rows corresponding to unique cell line IDs and columns corresponding to 22-digit BRD IDs. We excluded NAs and only included BRD ids/ drug names for drug columns.




LINCS Data


Phase I molecular signatures by Broad Institute (GSE92742_Broad_LINCS_Level5_COMPZ. MODZ_n473647x12328.gctx.gz) were used to determine gene expression caused by drug perturbation effects. We used this data for inferential statistics and to find drugs perturbing a set of genes.

Manual for Module 1 - Query a Gene

 Module 1: Finding a drug treatment with similar pertubation effects to a gene knockout

To start ‘Module 1: Query a Gene’, the user should enter an official Gene Symbol, such as CDK6, and select Cancer Type either "PanCan" (default option) or a specific cancer type (from the drop-down menu list). After which, the user should click on the orange button indicating [Submit] to initiate the query function.

Distributions of gene dependencies

The first function of the gene query module takes gene symbol as an input, examines the dependency score (gene effect) distribution across cancer cell lines from DepMap (n = 1054) and Sanger (n = 318) (Panel 1A, CRISPR loss-of-function screens). The density plot provides information of all gene effect scores (< -1 indicating strong dependency on the query gene in cells). A box plot (Panel 1B) is automatically generated in a pan-cancer manner that allows the user to visually examine cancer type associations from both Broad (light blue) and Sanger (light orange) DepMap datasets. On further examination of the box plot, user can observe that leukemia, myeloma and lymphoma show distinct dependency on CDK6. Note that cancer types at the bottom of box plot may not display due to the size of box plot in display. The user may resize the window to allow all label to be displayed or hover the cursor over the box plot to visualize the cancer type.

Gene-gene co-dependencies (similar gene knockouts)

Users can further explore the co-dependency of the query gene (in this case, CDK6) with all other ~17000 genes in both CRISPR screens. Distribution of the Pearson correlation coefficients are plotted in Panel 1C. A list of highly co-dependent genes, from both datasets, are reported in Panel 1D. For example, the top co-dependent genes to CDK6 include CCND3 and CCND2, both functioning as the regulators of CDKs. To better visualize the gene co-dependencies, an interactive network is displayed in Panel 1E on the query gene and its top 10 co-dependent genes (as ranked in Panel 1D; re-ranking Panel 1D by a column updates the network of Panel 1E). Here, the query gene is displayed in the center in blue circle. Connecting edges are color coded to reflect the direction (red: positive correlation, blue: negative correlation) and the width of an edge reflects the magnitude of the correlation coefficient from Broad DepMap screen.

Users can reorder/constrain the table ( Panel 1D) with respect to gene symbol or correlation coefficient values, via "Search" box, and sort each column, the network figure will update accordingly. A scatter plot (Panel 1F) between query gene and a top co-dependent gene will be plotted when the user selects a row (clicking the row from Panel 1D) to visually assess the robustness of co-dependency of two genes.

Descriptions to columns of Panel 1D:
  • Gene Name: Gene symbols
  • Corr in Broad: Correlation coefficients to the query gene in the Broad DepMap dataset. The width of the connecting edges in the network uses this column information.
  • Corr in Sanger: Correlation coefficients to the query gene from Sanger DepMap dataset.
  • Max Corr: Maximum correlation coefficient from Broad and Sanger DepMaps.
  • Avg Corr: Average correlation coefficient from Broad and Sanger DepMaps.

Gene-drug correlations (drug treatments similar to gene knockouts)

This section calculate Pearson correlation coefficient between gene dependency of the query gene to drug responses (both PRISM and GDSC drug screens) across PanCan collection or a selected cancer type. As we discuss in the overall manual of DepLink ("Help" from the top bar), PRISM screens measure the viability by fold-changes comparing to control treatment, and GDSC measures the IC50 of a drug. For the example of CDK6, four density distributions of correlation coefficients for CDK6 (assayed in Broad and Sanger DepMaps) to any drugs in the PRISM and GDSC are generated in Panel 1G, which are Broad DepMap vs PRISM, Broad DepMap vs GDSC, Sanger DepMap vs PRISM and Sanger DepMap vs GDSC. Panel 1H  shows a table of correlation coefficients for all four comparisons. Please note that the results of drugs assayed PRISM and GDSC are represented by different rows, with the first column denoting the corresponding drug screen.  

Similar to Panel 1E, Panel 1I displays the connected network between the query gene (CDK6, in blue circle) and top 10 correlated drugs (in orange diamond). Also, the connecting edges are coded with correlation coefficient from the Broad DepMap only. A scatter plot (Panel 1J) between query gene and a selected drug is generated when the user selects a row from  Panel 1H (clicking the row) to visually assess the robustness of gene-drug association using Broad and Sanger DepMap screens.

In the case of the querying CDK6, the top correlated drug to CDK6 knockout is the palbociclib from PRISM screen. Palbociclib is a CDK 4/6 inhibitor that blocks certain molecules involved in promoting the growth of cancer cells. CKD4/6 inhibitors are a class of prescription medicines that are used in combination with hormone therapies to treat adults with hormone receptor positive, HER3-negative advanced or metastatic breast cancer. As we illustrated in this module, this query method will enable users to discover drugs that potentially target the query gene.

Column description of table in Panel 1I,
  • Data Source: Drug screen data from PRISM or GDSC.
  • Broad ID:  Broad drug ID if available.
  • Drug Name: Common drug name if known.
  • Corr with Broad: Correlation coefficient of the query gene (from Broad DepMap CRISPR screen) to the drug screen data. Network connecting edge width uses this column.
  • Corr with Sanger: Correlation coefficient of the query gene (from Sanger DepMap CRISPR screen) to the drug screen data.
  • Max Corr: Maximum correlation coefficient between Corr with Broad and Corr with Sanger.
  • Avg Corr: Average correlation coefficient between Corr with Broad and Corr with Sanger.

Manual for Module 2: Query a Gene List

 Module 2: Finding a drug perturbing a set of genes

Given a short list of genes, 'Module 2: Query a Gene List' searches the molecular signatures for compounds that can up- or down-regulate the query genes. It analyzes the LINCS database to generate molecular signatures of 19811 compounds (i.e., changes in expression levels of 12328 genes following treatment) across pan-cancer cell lines.

To start ‘Module 2: Query a Gene List’, the user should enter a short list of official gene symbols separated by new lines (or click  [Example]), choose whether the genes are to be up- or down-regulated by a drug treatment, and click the orange  [Submit] button. Due to computational complexity, the module currently takes up to 10 core genes that the user expects a drug to perturb the most.

Pan-cancer essentialities of query genes

Panel 2A demonstrates the essentialities of the query genes to allow the user to assess the importance of the query genes on cancer cell viability. The density plot is displayed similarly to Panel 1A of Module 1 by Broad DepMap gene effect scores across all 1054 pan-cancer cell lines (< -1 indicating strong essentiality of a gene in a cell line).

Drugs perturbing query genes

The panel shows the perturbance of 19,811 compounds on the query genes. For the demonstrating example of selecting “down-regulated” and inputting CDK4, CDK6, and CCND1 in the text box, the panel searches all 19,811 molecular signatures for drugs with the ability of up-regulating the three cycle cell genes. Panel 2B provides results on both individual and summary of the three query genes. A series of statistical tests are performed to assess:

  • Is the perturbance statistical meaningful across cell lines for a gene?
  • Is the perturbance statistical meaningful across all query genes?
  • Is the perturbance on query genes significantly stronger than on other genes? This test is to assess the degree of unintended perturbance in genes that are not included in the user’s query list.

Descriptions of Panel 2B (user can click on the title of a column to sort compounds according to the column):

  • Name and ID: Drug name and Broad drug ID.
  • CDK4: Average change in CDK4 expression levels by a compound across cell lines. The change is represented by a z-score comparing post- and pre-treatment expression levels in each cell line and then averaged across cell lines treated with a compound in the LINCS project (n = 1-64 per drug). A positive value indicates upregulation of the gene after drug treatment.
  • CDK4 pVal: P-value of the changes in CDK4 expression levels. Significance is assessed by the one-tailed t-test of the expressional changes in CDK4 across cell lines against zero (i.e., no change). A left-tail test is performed when the user selects “down-regulated”.
  • CDK6, CDK6 pVal, CCND1, CCND1 pVal: Similar to CDK4.
  • Mean Perturbation Score: Average among the query genes (columns CDK4, CDK6, and CCND1).
  • T-test pVal: P-value to indicate whether the changes among all query genes are significantly above (up-regulated) or below (down-regulated) zero. A one-tailed t-test is performed based on the user’s selection of direction of regulation.
  • Permutation pVal: P-value to indicate whether the changes among all query genes are significantly stronger than on other genes. P-value is inferred by the permutation test of 100,000 random shuffles of genes for each drug (see Panel 2C for details).
  • Canonical SMILES: Canonical SMILES code of a drug that can be further analyzed in Module 4.
  • PubChem: Links to PubChem.
  • PRISM/GDSC: Indicator of whether the compound is tested in the two high-throughput drug screens. If yes, the user can navigate to Module 3 to further investigate the drug.

When the user selects a drug (a row) from Panel 2B, Panel 2C shows the empirical distribution that we generated to infer‘Permutation pVal’ for the corresponding drug.  For each drug, an empirical normal distribution is generated based on the mean and std of 100,000 random selections of the same number of genes as the query list from all 12328 genes included in LINCS molecular signatures. The vertical line indicates the observed mean perturbation score. Permutation pVal is estimated by the portion of random gene selections that achieve stronger mean perturbation scores than the one observed from query genes.

Manual for Module 3: Query a Drug

 Module 3: Finding a gene knockout with similar perturbation effects to a drug treatment

To start ‘Module 3: Query a Drug’, the user should enter a common drug name, such as palbociclib or a BRD ID, or select one from the drop-down menu and select a cancer type (or default option PanCan). If multiple instances are found in the two drug data sources, i.e. PRISM and GDSC, a table will pop up below the ‘Cancer Type’ drop-down menu and user should select the drug instance of interest by clicking on the row to start the analysis.

Distributions of drug sensitivities

Panel 3A examines the log-fold change (for PRISM drug) or IC50 (for GDSC drug) distribution across all cancer cell lines (PanCan) or for selected cancer type. The density plot provides information on the inhibition of cell viability induced by a drug. A box plot in Panel 3B is generated to allow the user to visually examine the effectiveness of the drug across all cancer types. Here, a more negative value indicates stronger inhibited cell vialibility following drug treatment and thus denote better sensitivity. Please note that cancer types at the bottom of boxplot may not display due to the size of box-plot in display. The user may resize the window to allow all label to be displayed or hover the cursor over the box-plot to visualize the cancer type.

Drug-drug co-sensitivities (similar drugs)

To explore the co-sensitivities of the query drug (for example, palbociclib in the PRISM dataset) with all other drugs in the same drug screening dataset. Distribution of the Pearson correlation coefficients are plotted in Panel 3C. Top co-sensitive drugs of the query are tabulated in Panel 3D. For instance, the top correlated drugs to palbociclib include ribociclib and abemaciclib which are also CDK inhibitors. To better visualize the drug co-sensitivities similar to Panel 1E in Module 1, an interactive network of the query drug and its top co-sensitive drugs is displayed in Panel 3E. Here the query drug palbociclib is displayed in the center in blue diamond. The connecting edges are color coded to reflect the direction (red: positive correlation, blue: negative correlation) and the width of the connecting edge reflects the magnitude of the correlation coefficient.

Users can constrain the table (in Panel 3D) using the ‘Search:’ box by drug name, Broad ID, or target genes for PRISM drugs and by drug name or Broad ID for GDSC drugs. Alternatively, the user can sort the table by clicking on the header of a table column. After reordering the table, the network figure (Panel 3E) will update automatically. A scatter plot (Panel 3F) between query drug and selected co-sensitive drug is plotted when the user selects a row (clicking the row), on the co-response table (Panel 3E), to visually assess the robustness of co-sensitive drugs.

Column descriptions of Panel 3D (for PRISM drug screen),
  • Drug Name: Common drug name.
  • Broad ID: Broad drug ID if available.
  • Correlation: Correlation coefficient of query drug to any other drugs in PRISM. Network edge width is based on this column.
  • Dose: Drug dose.
  • Screen ID: Drug screen ID.
  • Target Gene: Known target genes.
Column descriptions of Panel 3D (for GDSC drug screen),
  • Drug Name: Common drug name.
  • Broad ID: Broad drug ID if available.
  • Correlation: Correlation coefficient of query drug to other drugs in GDSC. Network edge width is based on this column.

Drug-gene correlations (similar gene knockouts)

Pearson correlation coefficients are automatically computed between the query drug and any gene dependencies in the two CRISPR screens (Broad and Sanger DepMaps) across PanCan collection or selected cancer type. Two density distributions of correlation coefficients of the query drug palbociclib are displayed in Panel 3G , i.e., palbociclib (PRISM) vs. Broad DepMap and palbociclib (PRISM) vs. Sanger DepMap. Panel 3H shows a table of correlation coefficients for both comparisons.

Panel 3I  displays the network visualization of correlations between the query drug (palbociclib, in blue diamond) and top 10 gene knockouts (in orange circle), where the connecting edges are coded for color and width based on correlation coefficient values. A scatter plot (Panel 3J) between the query drug and a selected gene is generated and updates itself when user selects a gene of interest from Panel 3H (by clicking a row) to visually assess the robustness of drug-gene association.

In case of query search for drug Palbociclib from the PRISM dataset, the top correlated genes to this drug are CDK6 and CCND1. In additional to this, palbociclib is found to achieve similar response to ribociclib and abemaciclib that are also CDK inhibitors. These results are in agreement to those of Module 1, where a query search on gene CDK6 listed ribociclib as one of the top correlation drug. To further investigate the similarity between palbociclib and these CDK inhibitors and other drugs, the user may use Module 4 by inputting the SMILES code of palbociclib.

Column descriptions of Panel 3H ,
  • Gene Name: Gene symbol.
  • Corr in Broad DepMap: Correlation coefficient of the query drug to a gene knockout from Broad DepMap dataset.
  • Corr in Sanger DepMap: Correlation coefficient of the query drug to a gene knockout from Sanger DepMap dataset.
  • Max Corr: Maximum correlation coefficient between Corr in Broad DepMap and Corr in Sanger DepMap.
  • Avg Corr: Average correlation coefficient between Corr in Broad DepMap and Corr in Sanger DepMap.

Manual for Module 4 - Query a New Drug

 Module 4: Finding a drug sharing similar biochemical fingerprints with a new compound

This module is designed to allow users to query new drugs by examining drug fingerprint similarity. To begin, the user should enter a canonical SMILE code of the compound of interest (either investigational or established), for example, the SMILES code for metformin:

      CN(C)C(=N)NC(N)=N

The algorithm then plots a 2D structure of the query drug in Panel 4A and generate corresponding biochemical fingerprints of 1047 bits using the R package rcdk:

    mols <- parse.smiles( drug_smiles )
    fps <- lapply(mols, get.fingerprint, type=c('pubchem', 'maccs') )

Here we have pre-calculated the fingerprints of 4686 drugs included in the PRISM project based on the canonical SMILES codes. The query drug is compared against all the 4686 compounds by the Tanimoto similarity, a commonly used  similarity measure fingerprints to assess structural and biochemical similarity. In Panel 4D, a table is populated to display top similar drugs to the query compound. For the query example, the module correctly hits metformin with Tanimoto similarity score of 1, followed by guanidine (where metformin is derived from) with a similarity of 0.489.

The user can select a drug of interest (a row) from the table to automatically generate Panels 4B and 4C . Panel 4B shows the 2D structure of the selected drug to be contrasted with that of the query drug in  Panel 4A. In  Panel 4C, a Venn diagram visualizes the overlap in biochemical features (1's in the fingerprints) between the query and selected drugs, providing additional information regarding the numbers of commonly shared structures and those unique to each drug.

We also pre-generated t-SNE low-dimensional representation of the biochemical fingerprints based on 4686 compounds of PRISM. In Panel 4E, the query drug (red dot) is projected to the 3D t-SNE plot by the average of the 3 most similar drugs (green crosses). The drug selected from Panel 4D is shown in a yellow diamond and all other PRISM drugs are labeled by grey dots. The selected drug (e.g., guanidine), as well as the query drug's 2D structure will be shown in the first 2 panels, allow users to inspect the similarity of these two selected drugs.
